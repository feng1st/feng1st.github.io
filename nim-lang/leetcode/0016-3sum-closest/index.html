[Nim4LeetCode] 16. 3Sum Closest

Posted on 2018, Mar 22 2 mins read

Problem

Given an array S of n integers, find three integers in S such that the sum is closest to a given number, target. Return the sum of the three integers. You may assume that each input would have exactly one solution.

    For example, given array S = {-1 2 1 -4}, and target = 1.

    The sum that is closest to the target is 2. (-1 + 2 + 1 = 2).

Solution

import algorithm

proc threeSumClosest(nums: openArray[int], target: int): int =
  let sorted = sorted(nums, system.cmp)
  result = sorted[0] + sorted[1] + sorted[2]
  for i in 0 .. sorted.len - 3:
    if i > 0 and sorted[i - 1] == sorted[i]:
      continue
    var
      j = i + 1
      k = sorted.len - 1
    while j < k:
      let sum = sorted[i] + sorted[j] + sorted[k]
      if sum == target:
        return sum
      if abs(sum - target) < abs(result - target):
        result = sum
      if sum < target:
        j += 1
      elif sum > target:
        k -= 1

when isMainModule:

  assert threeSumClosest([-20, -20, -20, -10, -10, 10, 10, 20, 20, 20], -61) ==
