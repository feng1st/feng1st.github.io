[Nim4LeetCode] 12. Integer to Roman

Posted on 2018, Mar 18 One min read

Problem

Given an integer, convert it to a roman numeral.

Input is guaranteed to be within the range from 1 to 3999.

Solution

const
  M = ["", "M", "MM", "MMM"]
  C = ["", "C", "CC", "CCC", "CD", "D", "DC", "DCC", "DCCC", "CM"]
  X = ["", "X", "XX", "XXX", "XL", "L", "LX", "LXX", "LXXX", "XC"]
  I = ["", "I", "II", "III", "IV", "V", "VI", "VII", "VIII", "IX"]

proc intToRoman(num: int): string =
  result = M[num div 1000] &
    C[(num mod 1000) div 100] &
    X[(num mod 100) div 10] &
    I[num mod 10]

when isMainModule:
  assert intToRoman(1) == "I"
  assert intToRoman(3999) == "MMMCMXCIX"
